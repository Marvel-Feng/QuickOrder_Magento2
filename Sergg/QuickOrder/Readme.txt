ТЗ
Написать модуль "Quick Order" который на странице продукта а также на странице категорий под каждым продуктом выводить кнопку Quick Order

При нажатии кнопки QuickOrder появляется popup форма с двумя обязательными полями Name, Phone, не обязательным Email и скрытым полем куда передается sku запрашиваемого продукта  и кнопка  Send

После отправки данные отправляются в отдельную таблицу

С Админки

Есть отдельный Grid который выводит список всех QuickOrder
В таблице присутствует поля  Sku продукта, Name, Phone, Email, Create At, Update At и  Status

Поле статус должно быть редактируемое с грида т.е можно было поменять статус с pending на approved

Статусы. Статусы для заказов админ может менять самостоятельно. Т.е есть дефолтный набор статусов который идет при установке с модулем допустим Pending, Approved и Decline. есть возможность выставить дефолтный статус при создании Quick Order например Pending. Администратор на отдельной странице может добавить новый статус, редактировать имя существующего, удалять статусы и менять дефолтный статус
